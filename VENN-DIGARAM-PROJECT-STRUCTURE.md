# Venn Diagram Project — Standard Color Mapping

Az alábbi színek egységesek minden SVG Venn diagram fájlban.
A betű (A, B, C, ...) mindig az adott Shape-hez tartozó standard színt kapja.

| Set | Hex Code  | Color Name   |
|-----|-----------|--------------|
| A   | `#FFF200` | Yellow       |
| B   | `#2E3192` | Blue         |
| C   | `#ED1C24` | Red          |
| D   | `#808285` | Grey         |
| E   | `#3C2415` | Brown        |
| F   | `#9E1F63` | Magenta      |
| G   | `#CA4B9B` | Pink         |
| H   | `#21AED1` | Cyan         |

## Megjegyzések

- A fenti hex kódok a nagybetűs formátumot használják — az SVG fájlokban előfordulhat kisbetűs változat is.
- A 6-set és 8-set fájlok D színe `#808285`, az 5-set Grünbaum D színe `#58595B` (eltérés, javítandó).
- Forgatás (rotate_labels.py) után a színeknek az új betűt kell követniük.
