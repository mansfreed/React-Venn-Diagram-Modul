# Get the vennDiagramLab package version

Returns the installed version of vennDiagramLab as a character string.
This trivial function exists so the Phase 0 skeleton has one public
export; Phase 1 introduces the real
[`analyze()`](https://zoliqua.github.io/Venn-Diagram-Lab/r/reference/analyze.md)
/ `load_*()` API.

## Usage

``` r
vdl_version()
```

## Value

Character string, the package version (e.g. "2.0.0").

## Examples

``` r
vdl_version()
#> [1] "2.4.2"
```
