# List bundled sample dataset names

Returns the names of the 5 bundled sample datasets, sorted
alphabetically. Use \[load_sample()\] to load one.

## Usage

``` r
list_samples()
```

## Value

Character vector of 5 sample identifiers.

## Examples

``` r
list_samples()
#> [1] "dataset_mock_gene_sets"              "dataset_mock_streaming_platforms"   
#> [3] "dataset_real_cancer_drivers_4"       "dataset_real_msigdb_cancer_pathways"
#> [5] "dataset_real_msigdb_immune_pathways"
```
